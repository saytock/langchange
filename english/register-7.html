DOWNLOAD SUPPORTED BROWSERS

Since Langchange is using the latest video chat technology, it will not run properly on browsers that are not supported.
Please use the supported browsers when you have talks at Langchange.

-For PC/Android Devices-
Use Google Chrome (free) or Firefox (free). Please download one if you haven't done.





-For iPhone/iPad(iOS)-
Use Google Chrome (free). Please download one if you haven't done.
"Appear.in" (free app for video chat) is also required for iOS users. Please download one in advance. The app will show up automatically when a talk starts.
